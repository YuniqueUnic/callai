# Code of Conduct

## Our pledge

We want callai collaboration to stay welcoming, constructive, and focused on shipping a useful tool.

## Expected behavior

- Be respectful and precise in technical feedback.
- Assume good intent; ask clarifying questions before escalating.
- Keep criticism about code, docs, and design decisions—not people.
- Credit others’ work and third-party licenses accurately.

## Unacceptable behavior

- Harassment, personal attacks, or discriminatory language
- Publishing private information without consent
- Spam, deliberate disruption, or bad-faith contributions

## Enforcement

Project maintainers may remove comments, close issues/PRs, or block contributors who violate this policy.

Report concerns to the repository owner via GitHub (`@YuniqueUnic`) or the contact email listed in `src-tauri/Cargo.toml` authors.
